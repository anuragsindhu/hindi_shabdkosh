## MacOS में वर्धा हिंदी शब्दकोश का अधिष्ठापन
* [वर्धा हिंदी शब्दकोश.dictionary.zip](binary/MacOS/वर्धा%20हिंदी%20शब्दकोश.dictionary.zip) को  डाउनलोड कर अन्‌जिप करें।
* MacOS पर शब्दकोश ऐप खोलें और फाइल मेनू से Dictionaries फोल्डर में जाएँ। शब्दकोश ऐप को बंद कर दें।
* वर्धा हिंदी शब्दकोश.dictionary को Dictionaries फोल्डर में रखें।
* शब्दकोश ऐप को पुनः खोलें और सुनिश्चित करें की **वर्धा हिंदी शब्दकोश** उपलब्ध शब्दकोशों की सूची में है।
